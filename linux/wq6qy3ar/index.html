grep命令

约 632 字大约 2 分钟

2025-05-27

grep (Global Regular Expression Print) 是一个强大的文本搜索工具，用于在文件中查找匹配特定模式的行。

1. 基本语法

grep [选项] 模式 [文件...]

1.1. 搜索控制选项

  • -i：忽略大小写
  • -v：反向匹配，显示不包含模式的行
  • -w：只匹配整个单词
  • -x：只匹配整行
  • -e 模式：指定多个模式
  • -f 文件：从文件中读取模式
  • -P：使用 Perl 兼容正则表达式 (PCRE)

1.2. 输出控制选项

  • -c：只显示匹配行的计数
  • -l：只显示包含匹配的文件名
  • -L：只显示不包含匹配的文件名
  • -n：显示匹配行的行号
  • -H：显示文件名（默认多个文件时）
  • -h：不显示文件名（默认单个文件时）
  • -o：只输出匹配的部分
  • -q：静默模式，不输出任何内容

1.3. 上下文控制

  • -A NUM：显示匹配行及其后 NUM 行
  • -B NUM：显示匹配行及其前 NUM 行
  • -C NUM：显示匹配行及其前后各 NUM 行

1.4. 文件控制

  • -r 或 -R：递归搜索目录
  • --include=GLOB：只搜索匹配 GLOB 的文件
  • --exclude=GLOB：跳过匹配 GLOB 的文件
  • --exclude-dir=DIR：跳过指定目录

2. 常用示例

  1. 基本搜索：
grep "pattern" file.txt
  1. 忽略大小写：
grep -i "pattern" file.txt
  1. 递归搜索目录：
grep -r "pattern" /path/to/dir
  1. 显示行号：
grep -n "pattern" file.txt
  1. 统计匹配行数：
grep -c "pattern" file.txt
  1. 显示匹配前后的内容：
grep -C 3 "pattern" file.txt  # 显示匹配行前后各3行
  1. 反向匹配：
grep -v "pattern" file.txt
  1. 只显示匹配部分：
grep -o "pattern" file.txt
  1. 使用正则表达式：
grep "^[A-Z]" file.txt  # 查找以大写字母开头的行
  1. 搜索多个文件：
grep "pattern" *.txt
  1. 从多个模式搜索：
# 这会查找文件中匹配 "模式1" 或 "模式2" 的行（相当于逻辑 OR 操作）。
# 也可以 grep "error\|warning" log.txt  # 使用 \| 转义
grep -e "pattern1" -e "pattern2" file.txt
  1. 搜索二进制文件中的文本：
grep -a "pattern" binaryfile
  1. 排除特定目录：
grep -r --exclude-dir=dir_to_exclude "pattern" /path
  1. 只列出包含匹配的文件名：
grep -l "pattern" *.txt
  1. 使用 Perl 正则表达式：
grep -P "\d{3}-\d{4}" file.txt  # 查找电话号码格式
  1. 用于从文件读取多个模式：
echo "error" > patterns.txt
echo "warning" >> patterns.txt
grep -f patterns.txt log.txt
  • 相当于 grep -e "error" -e "warning" log.txt。
  1. 指定多个文件：
grep "pattern" file1.txt file2.txt file3.log
