Mac 安装rabbitmq

约 92 字小于 1 分钟

mac

2023-04-09

1.使用brew安装mq

brew install rabbitmq

2.安装位置

/opt/homebrew/Cellar/rabbitmq

3.启动


brew servies start rabbitmq
# 进去目录
cd /opt/homebrew/Cellar/rabbitmq/3.11.11/sbin
# 执行脚本
./rabbitmq-server

cd /opt/homebrew/Cellar/rabbitmq/3.11.11/sbin \./rabbitmq-server

4.登陆

http://localhost:15672/#/
用户名：guest
密码：guest

5.关闭服务

brew services stop rabbitmq
