子网掩码

约 645 字大约 2 分钟

2025-05-24

子网掩码（Subnet Mask）是网络中的一个重要概念，用来划分IP地址的网络部分和主机部分。

1. IP地址的作用

  • 就像你的家庭地址（如北京市朝阳区XX街道1号），IP地址（如192.168.1.10）用来标识网络中的设备。
  • 但IP地址需要分成两部分：
    • 网络部分：代表整个子网（类似“朝阳区”）。
    • 主机部分：代表子网中的具体设备（类似“XX街道1号”）。

2. 子网掩码的功能

- 子网掩码是一串类似IP的数字（如255.255.255.0），它的作用是告诉设备如何区分IP地址中的网络和主机部分。
- 规则：子网掩码中255对应的部分是网络位，0对应的部分是主机位。
  - IP地址：192.168.1.10
  - 子网掩码：255.255.255.0
  - 这意味着前三个数字（192.168.1）是网络号，最后一个数字（10）是主机号。

3. 实际例子

- 如果两台设备的IP地址和子网掩码如下：
  - 设备A：192.168.1.10，子网掩码255.255.255.0
  - 设备B：192.168.1.20，子网掩码255.255.255.0
  - 判断是否在同一子网：比较网络部分（前三个数字192.168.1），相同则可以直接通信。
- 如果子网掩码是255.255.0.0：
  - 网络部分变为前两个数字（192.168），主机部分是后两个（1.10和1.20），此时它们仍在同一子网。

4. 为什么需要子网掩码？

  • 划分子网：比如公司将一个大网络分成多个小部门子网，提高效率。
  • 节省IP资源：避免给每个小网络分配独立的IP段。
  • 安全隔离：不同子网的设备不能直接通信，增强安全性。

5. 如何计算IP范围？

假设有一个IP地址 192.168.1.0/26（子网掩码 255.255.255.192），计算步骤如下：

步骤1：确定网络地址

- 将IP和子网掩码做按位与运算：
  - IP: 192.168.1.0 → 11000000.10101000.00000001.00000000   
  - 掩码: 255.255.255.192 → 11111111.11111111.11111111.11000000
  - 结果（网络地址）: 11000000.10101000.00000001.00000000 -> 192.168.1.0

步骤2：确定广播地址 - 将主机位全部置1： - 网络地址: 192.168.1.00000000（后6位是主机位） - 广播地址: 192.168.1.00111111 → 192.168.1.63。

步骤3：可用IP范围 - 起始IP：网络地址 +1 → 192.168.1.1 - 结束IP：广播地址 -1 → 192.168.1.62 - 可用IP数量：62个（即 192.168.1.1 ~ 192.168.1.62）
